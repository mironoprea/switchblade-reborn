# Switchblade Reborn

Standalone Windows control app for the Razer DeathStalker Ultimate Switchblade UI
(4" touch LCD + 10 LCD dynamic keys) — replaces discontinued Razer Synapse 2.0.

**Status: planning complete, implementation not started.**

Read in this order:

1. `RESEARCH.md` — protocol dossier and evidence (mandatory before coding)
2. `BUILD_SPEC.md` — the implementation spec; work it top to bottom, phase gates
   are hard gates
3. `PLAN.md` — strategy and rationale (background only)

Driver rollback instructions land here in Phase 0 (see BUILD_SPEC section 3).
