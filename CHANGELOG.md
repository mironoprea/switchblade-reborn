# Changelog

## 1.0.0

- Consolidated development onto one standalone direct-USB application.
- Added main-display and ten-key image rendering, HID key input, native
  brightness, profiles, actions, widgets, web control, tray lifecycle, logging,
  autostart, reconnect handling, and Windows installer automation.
- Removed the separate compatibility utility, proprietary-runtime backend, and
  hardware-capture artifacts.
